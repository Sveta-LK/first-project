GIT

pwd - узнать где сейчас (print working directory - “показать рабочую папку”)
cd - сменить директорию (change directory) 
cd github/open-source-project    -   переходим через несколько директорий
ls - отобразить содержимое директории (list directory contents)
ls -а  -  вывели список, в котором отображаются скрытые файлы ., .. и •git
touch my-new-file.txt  -  создали файл my-new-file.txt
touch test1/test2/test3/test11.txt - создали документ test11.txt по пути в папку test3
mkdir new-dir   -  создали директорию new-dir (если не с текущего места, тогда указываем папку и через слеш какой документ создаем)
mkdir -p dir1/dir-inside/dir-deeper-inside  -  создали структуру директорий (папки в папках)
mkdir Desktop/test3 - создали папку в любом месте, вне зависимости где сейчас (сначала где создать/потом какую папку)
ср - копировать (copy) 
ср что_копируем куда_копируем/ (можем несколько файлов сразу)
cp index.html style.css script.js src/   -  скопировали три файла (index.html, style.css и script.js) в папку src
mv - переместить (move)
mv table.csv ./very-important-files - 
cat - прочитать файл (текстовый) (concatenate and print — «объединить и распечатать»)
rm, rmdir, rm -r  - (remove directory — «удалить директорию»)
~ - домашняя директория (Символ «тильда»)
cd ../ - на папку выше
  - буфер, В буфере хранятся все команды, которые вызывались до этого. По их списку можно перемещаться. 
tab - автоматически дописывает не только команды, но и пути
cd U[Tab]  - ввели первую букву имени пользователя и нажали Tab имя папки Username подставится автоматически
/[Tab] - увидеть содержимое корневой директории (англ. root directory)
cd / - быстро перемещаться в корневую директорию
cat ~/.gitconfig - покажет текущую учетную запись (данные в папке гитконфиг)
git config --list  - покажет текущую учетную запись (данные в папке гитконфиг)
